策划书

意行

综述

项目概述：通过ai，大数据，为用户提供最方便，快捷，且舒适的旅游计划。

项目特点：

  1. 私人订制，根据每一位用户的自身需求，量身定制一体化的旅游规划

  2. 前期用户端不收取任何费用，无广告，更好的用户体验

  3. 客户端拥有集成式的服务，一站式服务解决用户的所有痛点

目标用户：18-30岁左右需要独特旅行体验的年轻用户群体

设计说明

应用类型：生活实用

后端：Chat-GPT Fine-tuning + embedding models

开发工具：Dart

美术风格定位：极简

产品介绍

整体框架：Flutter

功能模块说明：First Minimum Viable Product(MVP) - 旅行计划生成，计划查看页面（图文），用户登录系统

开发进度规划

前期准备：2023.5.20 - 2023.9.7

技术开发：2023.9.7 - 2024.1.1

功能性测试及调优：2024.1.1 - 2024.4.1

全面测试：2024.4.1-2024.6.30

运营方案

运营模式：用户端免费，向企业端抽取提成 （前期无盈利）

运营目标：推广，旅游必备

盈利模式及利润预估

核心盈利点：企业端抽成（后期）

次要盈利点：会员（后期）

预算需求

人员成本：暂无

后端成本：

Fine-tuning models

Model Training Usage
Ada $0.0004 / 1K tokens $0.0016 / 1K tokens
Babbage $0.0006 / 1K tokens $0.0024 / 1K tokens
Curie $0.0030 / 1K tokens $0.0120 / 1K tokens
Davinci $0.0300 / 1K tokens $0.1200 / 1K tokens

Embedding models

Model Usage
Ada v2 $0.0004 / 1K tokens
Ada v1 $0.0040 / 1K tokens
Babbage v1 $0.0050 / 1K tokens
Curie v1 $0.0200 / 1K tokens
Davinci v1 $0.2000 / 1K tokens

设备成本：服务器成本不详，Domain成本不详

其他费用：notion，学习资源（udemy）
