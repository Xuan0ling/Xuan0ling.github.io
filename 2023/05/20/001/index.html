Flutter 学习笔记

Flutter环境搭建攻略

1. 安装Flutter SDK

访问Flutter官网选择你的操作系统(Windows、macOS、Linux),并按照说明下载并安装Flutter SDK. 安装完成后,打开终端/命令行,运行:

如果显示Flutter版本信息,则安装成功。

2. 配置编辑器

你可以选择VS Code(推荐)、Android Studio 或者 IntelliJ作为Flutter开发的编辑器。
这里我们以VS Code为例。安装VS Code后,启动软件,按Ctrl+P,输入install,选择Extensions: Install Extensions,然后搜索并安装Flutter和Dart插件。

3. 安装Android Studio和SDK

虽然我们使用VS Code作为编辑器,但我们仍需要Android Studio来安装Android SDK,Flutter需要它来构建Android应用。
从Android Studio官网下载并安装Android Studio。安装完成后,启动它,然后继续安装Android SDK、Android SDK Command-line Tools和Android SDK Build-Tools。

4. 安装iOS依赖(可选,仅MacOS)

如果要开发iOS Flutter应用,还需要在Mac上安装Xcode和CocoaPods。你可以从Mac App Store安装Xcode,然后在终端中运行:

来安装CocoaPods。

5. 运行flutter doctor

在终端中运行flutter doctor命令,检查你是否安装了所有Flutter开发需要的依赖项。如果发现任何缺失的依赖项,请按照提示进行安装。
所有依赖安装完成后,运行 flutter doctor应该显示No issues found!。此时你的Flutter开发环境就搭建成功了!
